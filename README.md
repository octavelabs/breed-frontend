# Breed

A spiritual companion app built to help you stay consistent in your walk with God.

## Features

- **Community Building**: Connect with fellow believers and build meaningful relationships
- **Daily Edification**: Access daily spiritual content and devotionals
- **Forums & Communities**: Engage in discussions and share experiences
- **Discipleship Tools**: Resources for spiritual growth and development
- **Online Christian Courses**: Educational content for deepening faith
- **Testimonials**: Stories from the community
- **Responsive Design**: Optimized for mobile and desktop experiences

## Tech Stack

- **Framework**: Next.js 16.1.2
- **Frontend**: React 19.2.3
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **Icons**: Lucide React
- **Linting**: ESLint

## Getting Started

### Prerequisites

- Node.js (version 18 or higher)
- npm, yarn, or pnpm

### Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd breed
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   # or
   pnpm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   # or
   yarn dev
   # or
   pnpm dev
   ```

4. **Open your browser**

   Navigate to [http://localhost:3000](http://localhost:3000) to see the application.

## Available Scripts

- `npm run dev` - Start the development server
- `npm run build` - Build the application for production
- `npm run start` - Start the production server
- `npm run lint` - Run ESLint for code linting
